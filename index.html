Categorisation des langage de programmation ------------------------------------------- 1) Langage dynamique - Python - JavaScript - Ruby - PHP - Perl 2) Langage statique - C - C++ - Java - C# - Go 3 ) Langage orienté client - HTML - CSS - JavaScript - TypeScript 4) Langage orienté serveur - Python - Ruby - PHP - Java - Node.js 5) Langage bas niveau - Assembleur - C - C++ 6) Langage haut niveau - Python - Java - Ruby - JavaScript - PHP 7) Langage interprété - Python - JavaScript - Ruby - PHP - Perl 8) Langage compilé - C - C++ - Java - C# - Go 9) Langage fonctionnel - Haskell - Lisp - Erlang 10) Langage orienté objet paradigme(manière de coder) paradigme : programmation procédurale dry = don't repeat yourself programmation fonctionnelle(orienté objet) - Java - C# - Python - Ruby - C++ 11) Langage procédural - C - Pascal - Fortran - Ada 12) Langage de script - JavaScript - Python - Ruby - PHP - Bash 13) Langage de requête - SQL - GraphQL - SPARQL 14) Langage de balisage - HTML - XML - Markdown 15) Langage de programmation parallèle - MPI - OpenMP - CUDA 16) Langage de programmation logique - Prolog - Datalog 17) Langage de programmation déclaratif
